# Security Policy

## Reporting a Vulnerability

We take the security of SwarmCLI CD seriously. If you believe you have found a security vulnerability, please report it to us responsibly.

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please send an email to hello@eldara.io .

### Our Process

1. We will acknowledge receipt of your report as soon as possible.
2. We will investigate the issue and confirm the vulnerability.
3. We will work on a fix.
4. We will provide a timeline for the fix and public disclosure.

Thank you for helping keep SwarmCLI CD secure!
